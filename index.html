Velkommen til min portefølje

Ferdigheter

Jeg er en engasjert og ambisiøs person som ønsker å jobbe som frontend-utvikler. I løpet av det siste halvåret har jeg gjennomført et front-end kurs, og jeg er interessert i å skape og designe et brukervennlig og funksjonelt effektivt grensesnitt. Gjennom kurset har jeg lært om å designe med Figma og utvikle nettsider ved hjelp av HTML, CSS, JavaScript og React. Jeg ble kjent med hele prosessen fra idédugnad til lansering av nettsiden, samt bruken av Git/Github og SQL

Erfaring

Jeg har en bachelorgrad i informatikk fra Egypt. Jeg har arbeidserfaring som lærer. I tillegg har jeg vært aktiv som frivillig i en organisasjon for samfunnsutvikling.

Egenskaper

Jeg er en arbeidsom, aktiv og dedikert person med analytisk tenkning og kreativitet. Jeg liker å være sosial og samarbeidsvillig for å bli en del av et nettverk og integrere meg selv i samfunnet. Jeg kan jobbe både selvstendig og i gruppe.

Hobbyer

I fritiden liker jeg å lese bøker og gå på tur med venner. Jeg liker også å se fotballkamper med min mann og spille brettspill med mine barn. Jeg har nylig begynt å søke og lete etter nye designer av nettsider slik at jeg kan få kreative ideer og inspirasjon.

Her er noen av prosjektene mine som jeg gjorde i løpet av kurset

Oppgaveliste-Nettapplikasjon

JS - HTML - CSS

Jeg har laget min egen oppgaveliste-nettapplikasjon. Applikasjonen kan motta data fra brukeren for behandling og visning av disse dataene. Jeg brukte localStorage for å holde dataene.

Flyr i verdensrommet

JS - HTML - CSS

I dette prosjektet implementerte jeg muligheten til å flytte et bevegelig HTML-element i fire retninger opp, ned, venstre og høyre ved hjelp av piltastene, og brukeren har lov til å flytte elementet umiddelbart ved å klikke hvor som helst på siden.

Disney karakterer

JS - HTML - CSS

Jeg har lært om asynkron Javascript, og spesifikt om å bruke løfter og async/wait for å håndtere asynkron Javascript. I dette prosjektet øvde jeg på disse ferdighetene ved å lage en nettside som bruker Disney API som jeg hentet data fra.

Trommemaskinen

JS - HTML - CSS

Jeg konstruerte en trommemaskin ved å benytte den medfølgende lydfilen. Hver trommelyd er tilordnet til en knapp som spiller av lyden når den trykkes, samt en tastatursnarvei som også aktiverer lyden. Tastatursnarveien skal vises på knappen.

Venus

Teamprosjekt - JS - HTML - CSS

Siden er kodet med HTML og CSS, hovedsakelig, og vi brukte også JS. Vi brukte god, semantisk og ryddig HTML, samt bruk av flex-boks til layout i CSS.

De 10 bestselgende videospillene

Teamprosjekt - Styling pågår - JS - HTML - CSS

I dette prosjektet laget vi en nettside som viser de 10 bestselgende videospillene. Ved å bruke den oppgitte objektmatrisen i app.js-filen, genererte vi dynamisk innhold for hvert spill, inkludert all nødvendig informasjon som tittel, utgivelsesår, plattform, salgstall og tilhørende logoer.

Enkel kalkulator

Enkel kalkulator laget med React for grunnleggende matematiske operasjoner. Tilbyr brukervennlig grensesnitt med mulighet til å legge til, trekke fra, multiplisere og dele tall.

Harry potter API

React - JS - CSS

Prosjektet mitt bruker React og react-router-dom for å vise data fra Harry Potter API, med klikkbare elementer for detaljer. Jeg har inkludert feilhåndtering, stilisering med CSS og fastnavigasjon.
